===============================================================================

　デスクトップの一部を任意の色で塗りつぶしたようになるプログラム

===============================================================================


■ソフトの概要

　単色のウィンドウを最前面に表示して、
　あたかもデスクトップの一部を任意の色で矩形に塗りつぶしたように見せるプログラムです。

　画像・テキストを貼り付けられるようにしてみました。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。

　.NET Framework 4 を使用します。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーしてください。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■使い方

　起動するには WindRect.exe を実行してください。

　[WR]というタスクアイコンが表示されるので、右クリックして追加を選んでください。
　スクリーンの左上に黒い矩形のウィンドウが表示されます。

　表示されたウィンドウを右クリックするとメニューが表示されます。


　●右クリックメニュー

　　位置と色の変更 ... 位置と色の変更ウィンドウを開きます。
　　常に手前に表示 ... ウィンドウを最前面に表示します。
　　ドラッグで移動 ... ウィンドウをドラッグで移動できるようにします。
　　剥がす         ... ウィンドウを閉じます。
　　終了           ... プログラムを終了します。


　●位置と色の変更

　　Ｘ   ... ウィンドウ左上のＸ座標
　　Ｙ   ... ウィンドウ左上のＹ座標
　　幅   ... ウィンドウの幅
　　高さ ... ウィンドウの高さ

　　画像サイズに合わせる ... 画像を指定した場合、その画像のサイズを幅・高さに設定します。

　　Ｒ   ... ウィンドウの色（赤）
　　Ｇ   ... ウィンドウの色（緑）
　　Ｂ   ... ウィンドウの色（青）

　　画像のボタン ... ウィンドウに貼り付ける画像ファイルを選択します。
　　リセット     ... このウィンドウを開いたときの値に戻します。


■補足

　ウィンドウをダブルクリックするとテキストを編集出来ます。
